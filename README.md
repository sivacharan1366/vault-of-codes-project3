Internship Programs Webpage

A modern and visually appealing internship programs webpage created using HTML and CSS. This project is a creative redesign of a basic internship listing page with a premium dark theme, interactive animations, and responsive layout.

Features
Dark aesthetic UI
Responsive design for mobile and desktop
Animated aurora background effects
Glassmorphism-inspired internship cards
Smooth floating card animations
Interactive hover effects
Real working links for each internship category
No JavaScript used
Built entirely with HTML and CSS
